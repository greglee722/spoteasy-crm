🔌 Google Sheets Connection
Connect your Google Sheet to load and save lead data.
1
Create a Google Sheet
Create a new Google Sheet with these exact column headers in row 1:
Business Name, Business Type, URL, First Name, Location, Domain Rating, Instagram, IG Reach Out Date, Pipeline Stage, Last Email Sent, Last Updated, Notes
2
Make your Sheet public (read)
Share → Anyone with the link → Viewer. Required for the app to read data.
3
Get a Google Sheets API Key
Go to console.cloud.google.com → Enable Google Sheets API → Create an API Key. Required for writing data.
4
Enter Sheet ID & API Key below
Sheet ID is in the URL: docs.google.com/spreadsheets/d/SHEET_ID/edit
