🔌 Connect to Google Sheets
1
Create a Google Sheet
Create a new Google Sheet with these exact column headers in row 1:
Business Name, Business Type, URL, First Name, Location, Domain Rating, Instagram, IG Reach Out Date, IG Status, Pipeline Stage, Mock Up Sent Date, Follow Up Date 1, Follow Up Date 2, Last Follow Up Date, Notes
2
Publish your Sheet to the web
In Google Sheets: File → Share → Publish to web → Select your sheet → Publish as CSV. Copy the URL and paste it below.
3
Get a Google Sheets API Key
Go to console.cloud.google.com → Create project → Enable Google Sheets API → Create an API Key. This lets the app write data back to your sheet.
4
Enter your Sheet ID & API Key
Your Sheet ID is in the URL: docs.google.com/spreadsheets/d/SHEET_ID/edit
